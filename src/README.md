# Hotel REST API

Spring Boot application for managing hotel system.

## Features
- Room management
- Guest management
- Reservation system
- RESTful API (CRUD)
- Swagger documentation

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- MySQL / H2
- Swagger

## Run project
mvn spring-boot:run

## API docs
http://localhost:8080/swagger-ui.html
