# Hotel Service API

Spring Boot backend for hotel booking system.

## Features
- Guests management
- Hotels management
- Rooms management
- Reservations system
- Authentication (login/register)
- Cucumber BDD tests
- Unit tests (Mockito)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Cucumber
- JUnit / Mockito
